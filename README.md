# RWB Form Designer Mobile v1.7.4

A mobile-friendly visual form designer for Tkinter.

Created by Robert William Blennerhed  
Developed in collaboration with ChatGPT  
Project: RWB Tech Lab

---

## Overview

RWB Form Designer Mobile is a lightweight and touch-friendly tool for designing Tkinter interfaces on mobile devices.

It is specifically designed for environments such as:

- Android (Pydroid 3)
- Small screens
- Limited development setups

The goal is to make GUI design accessible anywhere — even on a phone.

---

## Mobile Focus

This version is optimized for:

- Touch interaction instead of mouse precision
- Smaller screen layouts
- Simplicity over complexity
- Fast experimentation

---

## Features

- Visual form design for Tkinter
- Tap to select components
- Drag to move widgets
- Resize widgets (mobile-adapted)
- Undo last action
- Lightweight and fast
- Designed for mobile Python environments
